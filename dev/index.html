GeometricAlgebra.jl

GeometricAlgebra.jl aims to provide a simple interface for geometric algebra (a.k.a. Clifford algebra) in the Julia language.

julia> using GeometricAlgebra

julia> x, y, z = basis((1, 1, 1)); # basis for the algebra of physical space

julia> a = x + 2y + 3z # vector
Grade-1 Multivector{⟨+++⟩, 1, Vector{Int64}}:
 1 v1
 2 v2
 3 v3

julia> R = exp(π/4*x*y) # rotor
MixedMultivector{⟨+++⟩, Vector{Float64}}:
 0.7071067811865476
 0.7071067811865475 v12

julia> R*a*~R # rotate the vector
MixedMultivector{⟨+++⟩, Vector{Float64}}:
 2.0 v1 + -0.9999999999999997 v2 + 3.0 v3

Quick start

Generate basis vectors for algebra of given signature

julia> basis((1, 1, 1)) # also `basis(3)`
3-element Vector{Blade{⟨+++⟩, 1, bits, Int64} where bits}:
 1v1
 1v2
 1v3

julia> basis((x=-1, y=-1, z=-1)) # labelled anti-Euclidean basis
3-element Vector{Blade{⟨x-,y-,z-⟩, 1, bits, Int64} where bits}:
 1x
 1y
 1z

julia> basis(OffsetSignature((t=-1, x=+1, y=+1, z=+1), 0:3)) |> sum
Grade-1 Multivector{⟨t-,x+,y+,z+⟩ with indices 0:3, 1, Vector{Int64}}:
 1 t
 1 x
 1 y
 1 z

Perform arithmetic on blades

julia> x*y*z # a grade 3 blade
Grade-3 Blade{⟨x+,y+,z+⟩, 3, 0b111, Int64}:
 1 xyz

julia> x - 3y # a vector, or grade 1 multivector
Grade-1 Multivector{⟨x+,y+,z+⟩, 1, Vector{Int64}}:
  1 x
 -3 y
  0 z

julia> (1 - 2x*y)/(1 + x*y) # an inhomogeneous multivector of mixed grade
MixedMultivector{⟨x+,y+,z+⟩, Vector{Float64}}:
 -0.5
 -1.5 xy

julia> ans^10
MixedMultivector{⟨x+,y+,z+⟩, Vector{Float64}}:
 97.375
 -7.40625 xy

Component access

julia> v = basis(3)
3-element Vector{Blade{⟨+++⟩, 1, bits, Int64} where bits}:
 1v1
 1v2
 1v3

julia> a = 1 + 2v[1] + 3v[1]v[2]+ 4v[1]v[2]v[3]
MixedMultivector{⟨+++⟩, Vector{Int64}}:
 1
 2 v1
 3 v12
 4 v123

julia> a[2,1] # access the yx = -xy component
-3

julia> a[] == scalar(a) == 1
true

julia> pseudoscalar(a)
ERROR: UndefVarError: pseudoscalar not defined

For labelled signatures such as (x=1, y=1, z=1), you can access components using the basis labels, too:

julia> x, y, z = basis((x=1, y=1, z=1))
3-element Vector{Blade{⟨x+,y+,z+⟩, 1, bits, Int64} where bits}:
 1x
 1y
 1z

julia> a = x - y
Grade-1 Multivector{⟨x+,y+,z+⟩, 1, Vector{Int64}}:
  1 x
 -1 y
  0 z

julia> a[:x]
1

julia> a[2]
-1

Especially in the theory of relativity, convention dictates that components of spacetime 4-vectors are numbered from zero. This is possible with an OffsetSignature:

julia> STA = OffsetSignature((+1,-1,-1,-1), 0:3)
⟨+---⟩ with indices 0:3
(pretty-printed OffsetSignature{(1, -1, -1, -1), 0:3}())

julia> v = basis(STA)
4-element OffsetArray(::Vector{Blade{⟨+---⟩ with indices 0:3, 1, bits, Int64} where bits}, 0:3) with eltype Blade{⟨+---⟩ with indices 0:3, 1, bits, Int64} where bits with indices 0:3:
 1v0
 1v1
 1v2
 1v3

julia> a = rand(UInt8, 4)'v
Grade-1 Multivector{⟨+---⟩ with indices 0:3, 1, Vector{Int64}}:
  89 v0
 198 v1
  74 v2
 130 v3

julia> a[0] # first ("time") component
89
